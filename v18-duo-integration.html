DUO grid

The IBM Grid has three screen width breakpoints. Each breakpoint allows up to a certain amount of columns:

  • sm breakpoint (0px+) = 2 columns
  • md breakpoint (640px+) = 4 columns
  • lg breakpoint (1056px+) = 8 columns

cols will wrap as the sum of col widths exceed that breakpoint's max column width.

Tip: Before specifying width classes, direct children will take full width of the container at every breakpoint to support mobile-first coding. This means that there is no point in applying the full width for small [.ibm-col-sm-2]

Create a full-width col and then multiple cols that split width over larger breakpoints:
